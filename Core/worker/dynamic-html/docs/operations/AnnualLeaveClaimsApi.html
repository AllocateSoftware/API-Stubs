submitAnnualLeaveClaim

Submit an leave (vacation) claim for a worker

URL

http://healthsuite.allocatesoftware.com/api/worker/v1/worker/{id}/leave

HTTP Method

POST

Response Type

AsyncResponse

Parameters

  • Path: id String

    ID of the worker

  • Body: LeaveClaim LeaveClaim(LeaveClaim)

    Leave claim for a worker
