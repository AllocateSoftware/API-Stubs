This is the API

cancelProposal

Cancel the proposal (and the booking, if it has been accepted)

URL

http://healthsuite.allocatesoftware.com/api/booking/v1/proposals/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    ID of the proposal

createProposal

Offer a worker for filling a duty

URL

http://healthsuite.allocatesoftware.com/api/booking/v1/vacancy/{vacancyId}/proposals

HTTP Method

POST

Response Type

String

Parameters

  • Path: vacancyId String

    The ID of the vacancy

  • Body: ProposalDetails ProposalDetails(ProposalDetails)

    Details of the proposal for filling the duty (worker)
