This is the API

cancelDuty

Cancel a duty. Also cancels any bookings (if present).

URL

http://healthsuite.allocatesoftware.com/api/duty/v1/duties/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    ID of the duty

cancelDutyBooking

Cancel a duty booking.

URL

http://healthsuite.allocatesoftware.com/api/duty/v1/duties/{id}/booking

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    ID of the duty

createDuty

Create a new Duty

URL

http://healthsuite.allocatesoftware.com/api/duty/v1/duties

HTTP Method

POST

Response Type

String

Parameters

  • Body: DutyDetails DutyDetails(DutyDetails)

    Duty to be sent to agency

sendDutyForFulfillment

Send a duty for fulfillment by contingent workforce (Bank, Cloudstaff, Agency)

URL

http://healthsuite.allocatesoftware.com/api/duty/v1/duties/{id}/sendTo/{destination}

HTTP Method

POST

Response Type

Parameters

  • Path: id String

    ID of the duty

  • Path: destination String

    Desination to send the duty
