This is the API

cancelDuty

Service that cancels certain duty.

URL

http://localhost/{version}/trustcode/{trustCode}/duty/{dutyId}/cancel

HTTP Method

POST

Response Type

Parameters

  • Path: version String

    API version

  • Path: trustCode String

    Trust code from BankStaff

  • Path: dutyId Integer

    ID of the duty associated with the DE contract

fillDuty

Service that will create new booking on HR side with data sent by DE Provider.

URL

http://localhost/{version}/trustcode/{trustCode}/duty/{dutyId}/booking

HTTP Method

POST

Response Type

BookingResponse

Parameters

  • Path: version String

    API version

  • Path: trustCode String

    Trust code from BankStaff

  • Path: dutyId Integer

    ID of the duty associated with the DE contract

  • Body: BookingInfo BookingInfo(BookingInfo)

getDutyDetails

Returns details about a duties.

URL

http://localhost/{version}/trustcode/{trustCode}/duty/

HTTP Method

GET

Response Type

Duty

Parameters

  • Path: version String

    API version

  • Path: trustCode String

    Trust code from BankStaff

  • Query: id List(Integer)

    IDs of the duties to return

  • Query: page Integer

    Result page

  • Query: pageSize Integer

    Count of records per page

updateContractStatus

Service that will be used by DE Provider to update the contract status, after DE contract is created/approved.

URL

http://localhost/{version}/trustcode/{trustCode}/duty/{dutyId}/contract/{contractId}/status/{statusId}

HTTP Method

PUT

Response Type

Parameters

  • Path: version String

    API version

  • Path: trustCode String

    Trust code from BankStaff

  • Path: dutyId Integer

    ID of the duty associated with the DE contract

  • Path: contractId String

    ID of the contract from DE Provider's system

  • Path: statusId ContractStatus
