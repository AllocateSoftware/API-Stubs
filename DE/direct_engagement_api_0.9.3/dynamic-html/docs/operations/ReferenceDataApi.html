This is the API

getAgenciesByIds

Returns an array of agencies

URL

http://localhost/{version}/trustcode/{trustCode}/referenceData/agency/

HTTP Method

GET

Response Type

List

Parameters

  • Path: version String

    API version

  • Path: trustCode String

    Trust code from BankStaff

  • Query: id List(Integer)

    Array of reference data IDs to return

  • Query: page Integer

    Result page

  • Query: pageSize Integer

    Count of records per page
