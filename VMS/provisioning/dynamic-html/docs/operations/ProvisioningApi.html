This is the API

provisionStructure

Provision the supplied structure.

URL

http://healthsuite.allocatesoftware.com/vms/provisioning/v1/provision/customer/{id}

HTTP Method

PUT

Response Type

Parameters

  • Path: id String

    ID of the customer to provision

  • Body: StructureProvision StructureProvision(StructureProvision)

    Structure of the Customer (trust). This call should be considered a 'set' - elements omitted that were present in previous provisionings should be removed.
