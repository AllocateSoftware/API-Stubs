offerWorker

Offer a worker for a booking

URL

http://healthsuite.allocatesoftware.com/vms/api/v1/bookingRequests/{bookingRequestId}/offers

HTTP Method

POST

Response Type

Parameters

  • Path: bookingRequestId String

    ID of the booking request to offer a worker for.

  • Body: PersonIdentifier PersonIdentifier(PersonIdentifier)

    Worker to be offered
