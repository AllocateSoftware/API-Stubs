This is the API

acceptAgencyOffer

Accept an agency offer for a worker.

URL

http://healthsuite.allocatesoftware.com/vms/api/v1/bookingRequests/{bookingRequestId}/offers/{offerId}/accept

HTTP Method

PUT

Response Type

String

Parameters

  • Path: bookingRequestId String

    ID of the booking request to accept.

  • Path: offerId String

    ID of the offer to accept. Depending on the VMS, this may be a unique ID, an index to locate an agency, or the agency ID itself.

cancelBooking

Cancel a duty booking previously accepted. This does not cancel any outstanding booking requests or duty cascades.

URL

http://healthsuite.allocatesoftware.com/vms/api/v1/bookings/{bookingId}

HTTP Method

DELETE

Response Type

Parameters

  • Path: bookingId String

    ID of the booking to cancel.

cancelBookingRequest

Cancel a previously requested booking. If the booking request has resulted in actual bookings, these will also be cancelled.

URL

http://healthsuite.allocatesoftware.com/vms/api/v1/bookingRequests/{bookingRequestId}

HTTP Method

DELETE

Response Type

Parameters

  • Path: bookingRequestId String

    ID of the booking request to cancel.
