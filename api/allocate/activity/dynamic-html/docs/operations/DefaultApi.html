getActivities

URL

http://healthsuite.allocatesoftware.com/api/v1/activities

HTTP Method

GET

Response Type

List

Parameters

    The code for the customer (trust) for which the activities should be returned

  • Query: format String

    Format for the output. TBD, but may be used to control the scope of information returned.
