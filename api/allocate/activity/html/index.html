Up 
get /activities
(getActivities)
Provide hours and assignment details for workers

Query parameters

customerCode (required)
Query Parameter — The code for the customer (trust) for which the activities should be returned default: null
format (optional)
Query Parameter — Format for the output. TBD, but may be used to control the scope of information returned. default: null

Return type

array[HoursAssignment]
