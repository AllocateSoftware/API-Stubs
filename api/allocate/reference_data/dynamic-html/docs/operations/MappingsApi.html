createMapping

Create a new entry in a mapping set

URL

http://healthsuite.allocatesoftware.com/api/v1/mappingSets/{setId}/mappings

HTTP Method

POST

Response Type

String

Parameters

  • Path: setId String

    The ID of the mapping set

  • Body: Mapping Mapping(Mapping)

    Details of the mapping

http://healthsuite.allocatesoftware.com/api/v1/mappingSets/{id}/mappings/{mappingId}

HTTP Method

DELETE

Response Type

Parameters

    The ID of the vocabulary

    The ID of the mapping
