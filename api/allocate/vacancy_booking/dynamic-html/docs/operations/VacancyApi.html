This is the API

createProposal

Offer a worker for filling a duty

URL

http://healthsuite.allocatesoftware.com/api/v1/vacancies/{vacancyId}/proposals

HTTP Method

POST

Response Type

CreateProposalResponse

Parameters

  • Path: vacancyId String

    The ID of the vacancy

  • Body: ProposalDetails ProposalDetails(ProposalDetails)

    Details of the proposal for filling the vacancy (worker, costs, etc)

createVacancy

Create a new vacancy

URL

http://healthsuite.allocatesoftware.com/api/v1/vacancies

HTTP Method

POST

Response Type

CreateVacancyResponse

Parameters

  • Body: Vacancy Vacancy(Vacancy)

    Details of the vacancy
